  1. de event listtener wordt gekoppeld aan de slider: let sliders = document.getElementsByClassName("slider"); sliders[0].addEventListener(...)
  2. waar wordt de rode kleur van het blokje ingesteld: colorDemos[0].style.backgroundColor="red";
  3. The [0] indicates that we are accessing the first element (index 0) of the array. In JavaScript, arrays are zero-indexed, meaning the first element is at index 0, the second at index 1, and so on. In summary, the line of code sliders[0].addEventListener("change", update); adds an event listener to the first slider in an array, so that when the slider value changes, the update function is executed.
