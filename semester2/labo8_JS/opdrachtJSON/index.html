Vraag 1:

Het formaat dat wordt gebruikt om het Date object op te slaan in JSON is een ISO 8601 datumtijd formaat: "YYYY-MM-DDTHH:mm:ss.sssZ".

Vraag 2:

Wanneer een object meerdere keren voorkomt tijdens het opbouwen van een JSON-tekst, wordt het object niet herhaaldelijk in de tekst opgenomen. In plaats daarvan wordt het object slechts één keer opgenomen, en voor de herhaalde keren wordt er naar datzelfde object verwezen met een referentie.

Dit betekent dat als je bijvoorbeeld een object a hebt met een eigenschap b die verwijst naar een ander object c, en als c weer een eigenschap heeft die verwijst naar a, dan zal de JSON-tekst deze ver wijzingen correct behandelen zonder in een oneindige lus te belanden. Het zal het object c slechts één keer opnemen en vervolgens voor de tweede keer dat c wordt genoemd, zal het simpelweg verwijzen naar de eerdere vermelding. Dit maakt JSON-teksten efficiënter en voorkomt redundantie.
